# typing-text
Type Writer effect on any HTML Element

## usage
```
// get the element
const textElement = document.querySelector('.typing-text');

// make a string array
const devsLifecycle = [
    "eat.",
    "sleep.",
    "code.",
    "repeat.",
];

// start typing effect
setTyper(textElement, devsLifecycle);
```
Checkout the demo here: [demo](https://varunpvp.github.io/typing-text/)
